### 题目
[344. Reverse String](https://leetcode.com/problems/reverse-string/)

### 题目描述
```翻转字符数组```
### 思路
**对撞指针**swap
### 代码
```java
class Solution {
    public void reverseString(char[] s) {
        int l = 0;
        int r = s.length-1;
        while (l < r){
            swap (s, l, r);
            l++;
            r--;
        }
    }
     void swap (char[] s, int i, int j){
        char tmp = s[i];
        s[i] = s[j];
        s[j] = tmp;
     }
}
```
