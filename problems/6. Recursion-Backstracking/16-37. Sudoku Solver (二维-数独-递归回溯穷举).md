### 题目
[37. Sudoku Solver](https://leetcode.com/problems/sudoku-solver/)
### 题目描述
```
Write a program to solve a Sudoku puzzle by filling the empty cells.

A sudoku solution must satisfy all of the following rules:

Each of the digits 1-9 must occur exactly once in each row.
Each of the digits 1-9 must occur exactly once in each column.
Each of the the digits 1-9 must occur exactly once in each of the 9 3x3 sub-boxes of the grid.
Empty cells are indicated by the character '.'.

A sudoku puzzle...

...and its solution numbers marked in red.

Note:

The given board contain only digits 1-9 and the character '.'.
You may assume that the given Sudoku puzzle will have a single unique solution.
The given board size is always 9x9.
```

### 思路

### 关键点分析
#### 如何判断满足子数独限制？
关键是确定子数独位置。起始行为`row / 3 * 3`，起始列为`col /3 * 3`，遍历接下来的 9 个位置，则第 n 个位置坐标为（startRow + n / 3，startCol + n % 3）。

每 3 个为一组，组内 col + 1，组外 row + 1

eg. 判断 9 是否可以放在 board[4][5] 位置上

* 行：遍历 board[4][0~8] 看有没有 9
* 列：遍历 board[0~8][5] 看有没有 9
* **子数独**：遍历 board[4 / 3 * 3 + n / 3, 5 /3 * 3 + n % 3] 看有没有 9

```java
    private boolean isValide (char[][] board, int row, int col, char ch){
        int blkRowStart = row / 3 * 3;
        int blkColStart = col /3 * 3;
        for(int i = 0; i < 9; i++){
            // check 所在行
            if (board[row][i] == ch){
                return false;
            }
            // check 所在列
            if (board[i][col] == ch){
                return false;
            }
            // check 所在子数独
            if (board[blkRowStart + i / 3][blkColStart + i % 3] == ch){
                return false;
            }
        }
        return true;
    }
```

### 代码
```java 
package 递归_回溯;

public class _37_数独 {
    public void solveSudoku(char[][] board) {
        if (board == null || board.length != 9 || board[0].length != 9){
            return;
        }
        solveSudoku(board, 0, 0);
    }

    /**
     * 从 (row, col) 开始递归回溯穷举合适数字
     * @param board
     * @param row
     * @param col
     */
    private boolean solveSudoku (char[][] board, int row, int col){
        int maxCol = 9;
        int maxRow = 9;
        if (row >= maxRow){// 到最后一行下一行，结束
            return true;// 正确道路最底层 true
        }else if (col >= maxCol){// 换行
            return solveSudoku(board, row + 1, 0);
        } else {// 未到行尾、列尾
            if (board[row][col] != '.'){
                return solveSudoku(board, row, col + 1);
            }
            // 递归逻辑
            for (char ch = '1'; ch <= '9'; ch++){
                if (isValide (board, row, col, ch)){
                    // 假设
                    board[row][col] = ch;
                    if (solveSudoku(board, row, col + 1)){
                        // 若能走通，就一直返回 true，直到出去
                        return true;// 错误道路千千万，正确道路只有一条
                    }
                    // 若走不通，回溯，再试别的路
                    board[row][col] = '.';
                }
            }
        }
        return false;
    }

    /**
     * 在board[][] 中(row, col)位置判断数字ch是否满足限制
     * @param board
     * @param row
     * @param col
     * @return
     */
    private boolean isValide (char[][] board, int row, int col, char ch){
        int blkRowStart = row / 3 * 3;
        int blkColStart = col /3 * 3;
        for(int i = 0; i < 9; i++){
            // check 所在行
            if (board[row][i] == ch){
                return false;
            }
            // check 所在列
            if (board[i][col] == ch){
                return false;
            }
            // check 所在子数独
            if (board[blkRowStart + i / 3][blkColStart + i % 3] == ch){
                return false;
            }
        }
        return true;
    }
}
```
